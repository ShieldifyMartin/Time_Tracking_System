## Features
- Админска и стандартна функционалност
- Графици с календар (напр. FullCalendar).
- Визуализации на данни чрез графики (напр. Chart.js).
- Автоматично известяване за закъснения или превишено работно време.
- Генериране на PDF отчети с dompdf.

## Local Setup

1. `npm run dev`
2. `php artisan serve`
